# 依赖分析

本文档详细分析了项目 `package.json` 中的所有依赖包及其用途。

## 生产依赖 (Dependencies)

### 核心框架

#### react (^19.0.0)
- **描述**: React 核心库，用于构建用户界面的 JavaScript 库
- **用途**: 项目的核心框架，提供组件化开发能力

#### react-dom (^19.0.0)
- **描述**: React 的 DOM 渲染器
- **用途**: 将 React 组件渲染到浏览器 DOM 中

### 路由

#### react-router (^7.1.5)
- **描述**: React 官方路由库
- **用途**: 处理单页应用的路由导航和页面跳转

### 样式与 UI

#### tailwind-merge (^3.0.1)
- **描述**: 智能合并 Tailwind CSS 类名的工具库
- **用途**: 解决 Tailwind CSS 类名冲突，合并条件类名

#### clsx (^2.1.1)
- **描述**: 条件类名构建工具
- **用途**: 根据条件动态构建 CSS 类名字符串

### 图表与可视化

#### apexcharts (^4.1.0)
- **描述**: 现代化的 JavaScript 图表库
- **用途**: 创建各种类型的数据可视化图表（折线图、柱状图等）

#### react-apexcharts (^1.7.0)
- **描述**: ApexCharts 的 React 封装
- **用途**: 在 React 组件中使用 ApexCharts 图表

#### @react-jvectormap/core (^1.0.4)
- **描述**: React 矢量地图核心库
- **用途**: 提供矢量地图的基础功能

#### @react-jvectormap/world (^1.1.2)
- **描述**: 世界地图数据包
- **用途**: 在应用中显示世界地图

### 日历组件

#### @fullcalendar/core (^6.1.15)
- **描述**: FullCalendar 核心库
- **用途**: 提供日历功能的基础 API

#### @fullcalendar/react (^6.1.15)
- **描述**: FullCalendar 的 React 集成
- **用途**: 在 React 中使用 FullCalendar 组件

#### @fullcalendar/daygrid (^6.1.15)
- **描述**: FullCalendar 日网格视图插件
- **用途**: 显示月视图的日历网格

#### @fullcalendar/timegrid (^6.1.15)
- **描述**: FullCalendar 时间网格视图插件
- **用途**: 显示时间轴视图的日历

#### @fullcalendar/list (^6.1.15)
- **描述**: FullCalendar 列表视图插件
- **用途**: 以列表形式显示日历事件

#### @fullcalendar/interaction (^6.1.15)
- **描述**: FullCalendar 交互功能插件
- **用途**: 支持拖拽、点击等日历交互操作

### 表单组件

#### flatpickr (^4.6.13)
- **描述**: 轻量级、功能强大的日期时间选择器
- **用途**: 提供日期和时间选择功能

### 交互功能

#### react-dnd (^16.0.1)
- **描述**: React 拖放库
- **用途**: 实现拖拽功能（如日历事件的拖拽）

#### react-dnd-html5-backend (^16.0.1)
- **描述**: React DnD 的 HTML5 后端
- **用途**: 为 react-dnd 提供基于 HTML5 的拖放支持

#### react-dropzone (^14.3.5)
- **描述**: React 文件拖放上传组件
- **用途**: 实现文件拖放上传功能

#### swiper (^11.2.3)
- **描述**: 现代化的触摸滑块/轮播图库
- **用途**: 创建轮播图、幻灯片等滑动组件

### 工具库

#### react-helmet-async (^2.0.5)
- **描述**: 文档头管理库（异步版本）
- **用途**: 动态管理 HTML 文档的 head 标签（title、meta 等）

## 开发依赖 (DevDependencies)

### 构建工具

#### vite (^6.1.0)
- **描述**: 下一代前端构建工具
- **用途**: 提供快速的开发服务器和生产构建

#### @vitejs/plugin-react (^4.3.4)
- **描述**: Vite 的 React 插件
- **用途**: 在 Vite 中支持 React 开发（热更新、JSX 转换等）

#### vite-plugin-svgr (^4.3.0)
- **描述**: Vite 的 SVG 转 React 组件插件
- **用途**: 将 SVG 文件作为 React 组件导入使用

### 样式处理

#### tailwindcss (^4.0.8)
- **描述**: 实用优先的 CSS 框架
- **用途**: 提供原子化 CSS 类名，快速构建 UI

#### @tailwindcss/postcss (^4.0.8)
- **描述**: Tailwind CSS 的 PostCSS 插件
- **用途**: 在 PostCSS 中集成 Tailwind CSS

#### postcss (^8.5.2)
- **描述**: CSS 后处理器
- **用途**: 处理 CSS 转换和优化

### TypeScript 支持

#### typescript (~5.7.2)
- **描述**: TypeScript 编程语言
- **用途**: 提供类型检查和 TypeScript 编译

#### @types/react (^19.0.12)
- **描述**: React 的 TypeScript 类型定义
- **用途**: 为 React 提供 TypeScript 类型支持

#### @types/react-dom (^19.0.4)
- **描述**: React DOM 的 TypeScript 类型定义
- **用途**: 为 React DOM 提供 TypeScript 类型支持

### 代码质量

#### eslint (^9.19.0)
- **描述**: JavaScript/TypeScript 代码检查工具
- **用途**: 检查代码质量，发现潜在问题

#### @eslint/js (^9.19.0)
- **描述**: ESLint 的 JavaScript 配置
- **用途**: 提供 ESLint 的基础配置

#### typescript-eslint (^8.22.0)
- **描述**: TypeScript 的 ESLint 解析器
- **用途**: 让 ESLint 支持 TypeScript 代码检查

#### eslint-plugin-react-hooks (^5.0.0)
- **描述**: React Hooks 的 ESLint 规则
- **用途**: 检查 React Hooks 的使用是否正确

#### eslint-plugin-react-refresh (^0.4.18)
- **描述**: React Fast Refresh 的 ESLint 规则
- **用途**: 确保组件可以安全地进行 Fast Refresh

#### globals (^15.14.0)
- **描述**: 全局变量定义库
- **用途**: 为 ESLint 提供全局变量定义

## 依赖覆盖 (Overrides)

项目配置了以下依赖覆盖，以解决对等依赖冲突：

- **react-helmet-async**: 支持 React 16.8+、17、18、19
- **@react-jvectormap/core**: 支持 React 16.8+、17、18、19
- **@react-jvectormap/world**: 支持 React 16.8+、17、18、19

这些覆盖确保了相关库与 React 19 的兼容性。

## 总结

### 主要功能模块

1. **UI 框架**: React 19 + TypeScript
2. **样式系统**: Tailwind CSS 4.0
3. **路由**: React Router 7
4. **图表**: ApexCharts
5. **日历**: FullCalendar
6. **地图**: React JVectorMap
7. **构建工具**: Vite 6
8. **代码质量**: ESLint + TypeScript

### 项目特点

- 使用最新的 React 19 和 Tailwind CSS 4.0
- 完整的 TypeScript 支持
- 丰富的 UI 组件库（图表、日历、地图等）
- 现代化的构建工具（Vite）
- 完善的代码质量检查工具
